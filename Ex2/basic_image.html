Image Formats and Attributes

What is the difference between the real size of the image and this attributes?

The actual size of the image may be different from the dimensions specified in the width and height attributes. These attributes control the visual size of the image on the web page but do not change the intrinsic dimensions of the image file. Therefore, adjusting these attributes can affect the aspect ratio and visual appearance of the image.
